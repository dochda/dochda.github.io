07 — Interaction

Custom Cursor

Gold-Punkt als Cursor. Default: 8px. Hover auf Links/Buttons: 40px, 50% opacity.

Default — 8×8px, #C8A147
Hover — 40×40px, 50% opacity
CSS Snippet
body { cursor: none; }
#cursor {
  position: fixed;
  width: 8px; height: 8px;
  background: #C8A147;
  border-radius: 50%;
  pointer-events: none;
}
#cursor.hover {
  width: 40px; height: 40px;
  opacity: 0.5;
}
